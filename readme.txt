Trabajo Práctico Entregable Nº1

Trabajo requerido: 
    • Dibujar figuras de diferente tamaño en posiciones al azar del canvas utilizando
      clases y objetos. Las figuras pueden ser círculos, rectángulos, etc. 
    • Implementar funcionalidad para verificar si un punto se encuentra dentro de una
      figura. Usar la posición del mouse como punto a verificar. Agregar los métodos
      correspondientes al diseño del ejercicio anterior.
    • Implementar la funcionalidad necesaria para mover las figuras dentro del canvas:
      a. Mover una figura seleccionada utilizando el mouse (arrastrar) requerido. 
      b. Mover una figura seleccionada utilizando el teclado requerido.

Modalidad de entrega:
    Correo electronico interfaces-tudai@alumnos.exa.unicen.edu.ar o por Moodle en el
    recurso tarea asignado para el trabajo entregable 1, el formato esperado es enlace git o
    apellido_nombre.ZIP

Fecha de entrega 31/08

Para comenzar a dibujar las figuras se debe presionar el boton "Agregar figuras".

Para arrastrar una figura con el mouse se debe hacer click y manteniendolo presionado moviendo el mouse.

Para arrastrar una figura con las flechas se debe seleccionar la figura con el mouse y
 manteniendo el click presionado se pueden mover con las flechas.
